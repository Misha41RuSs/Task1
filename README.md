# 🎨 Рисование фигур в JavaFX

[![Javadoc](https://img.shields.io/badge/JavaDoc-Online-blue)](https://Misha41RuSs.github.io/Task1/)

Интерактивное приложение для рисования простых геометрических фигур на холсте с помощью JavaFX. Включает в себя поддержку генерации фигур по числу сторон (Многоугольник) в соответствии с вариантом задания.


---

## 🧱 Главное окно приложения

Главное окно состоит из:

- Верхней панели с заголовком, выбором цвета, толщины линий и режима заливки
- Центральной части с холстом (`Canvas`)
- Нижней панели с кнопками для выбора режима рисования и управления

![Главное окно](https://i.postimg.cc/pTyMNPgc/image.png)

---

## ✍️ Поддерживаемые режимы рисования

Вы можете выбрать один из следующих режимов:

- ✅ **Прямоугольник** — 2 клика: первая точка — верхний левый угол, вторая — нижний правый
- ✅ **Треугольник** — 3 клика: каждая вершина
- ✅ **Окружность** — 2 клика: центр и точка на окружности (рассчитывается радиус)
- ✅ **Многоугольник** — 2 клика: центр и крайняя точка. Количество сторон задается в специальном поле.
- ✅ **Свободное рисование** — зажмите мышь и рисуйте линию от руки
- 🧽 **Очистить** — удаляет все с холста
- ↩️ **Отменить** — удаляет последнюю нарисованную фигуру

## 🛠️ Управление

| Элемент управления | Описание |
|--------------------|----------|
| 🎨 Цвет            | Выбор цвета линий/заливки |
| 📏 Толщина         | Толщина линий от 1 до 10 |
| 🪣 Заливка         | Включает заливку фигур |
| 📋 Режим           | Выпадающий список: прямоугольник, треугольник, окружность, свободное рисование |
| 🧼 Очистить        | Очищает холст |
| ↩️ Отменить        | Отменяет последнее действие (фигуру) |

## 📦 Структура проекта

Архитектура проекта разделена на логические пакеты для соблюдения принципов ООП:

```text
src/main/
├── java/org/swe/figures/
│   ├── Model/        # Классы графических фигур (Shape, Circle, NPolygon, Фабрика фигур)
│   ├── Tools/        # Инструменты рисования и их фабрика (DrawTool, ShapeTool, NPolygonTool)
│   ├── View/         # Классы для сохранения/загрузки состояния (паттерн Memento: CanvasState)
│   ├── Controller.java # Контроллер для связывания логики и пользовательского интерфейса
│   └── Main.java     # Точка входа в приложение
└── resources/org/swe/figures/
    └── main-form.fxml # FXML-разметка графического интерфейса
```

## 🚀 Как запустить

Проект использует систему сборки **Maven**. Для работы требуется установленная Java версии 23 (или совместимая).

**Для запуска приложения через терминал выполните:**
```bash
mvn clean javafx:run
```

**Для генерации Javadoc-документации локально:**
```bash
mvn clean javadoc:javadoc -B
```
Сгенерированная документация будет доступна в папке `target/site/apidocs/index.html`.
